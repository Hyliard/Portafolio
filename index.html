Cifrado Cesar de Luis

Esta aplicación web permite cifrar texto utilizando el algoritmo del Cifrado César, un método clásico de criptografía. Los usuarios pueden ingresar un texto y especificar un valor de desplazamiento, y la aplicación genera el texto cifrado en tiempo real. La aplicación presenta una interfaz limpia y moderna que facilita la entrada de texto y el ajuste del desplazamiento.
Al hacer clic en el botón "Cifrar", el texto ingresado se transforma inmediatamente, permitiendo a los usuarios ver el resultado al instante.
Se asegura que el desplazamiento esté dentro de un rango válido (1-25) para evitar errores.
Diseñada con una paleta de colores suaves y un diseño responsivo, la aplicación ofrece una experiencia visual agradable.

Este proyecto fue creado para practicar mis habilidades en desarrollo web y programación, así como para explorar conceptos de criptografía básica. Además, me permitió aplicar principios de diseño de interfaz y mejorar mi comprensión de la interacción del usuario con aplicaciones web.
